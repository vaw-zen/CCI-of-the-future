## 🛠️ Getting Started

### Prerequisites
```bash
Node.js 18+ 
npm or yarn
Gmail account with app password
Google Analytics & Search Console accounts
```

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/vaw-zen/CCI-of-the-future.git
   cd CCI-of-the-future
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment setup**
   ```bash
   cp .env.example .env.local
   # Add your credentials:
   # GMAIL_USER, GMAIL_PASS, SUPABASE_URL, etc.
   ```

4. **Run development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open in browser**
   ```
   http://localhost:3000
   ```
